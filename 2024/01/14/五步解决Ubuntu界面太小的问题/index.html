对于20版本及以上的unbuntu我们可以通过安装open-vm-tools来解决界面大小的问题，具体步骤如下：

1、首先打开终端，输入命令sudo apt-get update，并输入你的用户密码，更新软件包。

该命令是用于更新Ubuntu系列Linux操作系统的软件包列表的命令。

2、更新完软件包后，输入sudo su ，进入超级用户模式（root用户权限）。

该命令用于切换到超级用户(root用户)的权限

3、执行apt-get install open-vm-tools命令，正常情况会出现以下类似以下界面的情况，那么根据提示输入y，之后回车确认，等待安装。

该命令的作用是安装 open-vm-tools 软件包

4、继续执行apt-get install open-vm-tools-desktop

open-vm-tools-desktop是open-vm-tools 的一个扩展

5、上述都安装好后，输入reboot命令，进行重启

reboot为重启命令，用于重启计算机。

重启后，就能发现Ubuntu已适配客户机的窗口大小。
